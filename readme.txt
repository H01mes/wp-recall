WordPress plugin to create a personal user profile

== Description ==

WordPress plugin to create a personal user profile. The plugin creates a personal account of the user, user group, reviews. The plugin allows you to exchange private messages and more. Plug-in extends its functionality through its supplements.

== Installation ==

1. Upload dir `recall` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
